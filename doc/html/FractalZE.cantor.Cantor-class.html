Class Cantor

source code

Fractal de Cantor. Construccion:

  1. Se parte de un segmento.
  2. Se lo divide en 3 partes iguales.
  3. Se descarta la del medio.
  4. Se repite esto sucesivamente con cada segmento las veces que se desee.
